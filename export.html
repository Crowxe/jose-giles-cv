Cómo exportar tu CV a PDF

Método 1: Imprimir a PDF desde tu navegador

  1. Abre el archivo index.html en tu navegador.
  2. Presiona Ctrl+P (Windows/Linux) o Cmd+P (Mac) para abrir el diálogo de impresión.
  3. En el menú desplegable de "Destino" o "Impresora", selecciona "Guardar como PDF".
  4. Asegúrate de que la opción "Gráficos de fondo" esté activada (en Chrome, está en "Más configuración").
  5. Haz clic en "Guardar" y elige dónde guardar el archivo PDF.
Abrir el CV

Método 2: Usar una herramienta online

  1. Abre el archivo index.html en tu navegador.
  2. Asegúrate de que se vea correctamente y que estés satisfecho con su apariencia.
  3. Ve a Web2PDF Converter o PDFCrowd.
  4. Copia la URL completa de tu archivo HTML local o sube el archivo HTML directamente.
  5. Configura las opciones de conversión según tus preferencias.
  6. Haz clic en "Convertir" y descarga el PDF generado.

Método 3: Usar el script de conversión (para usuarios técnicos)

  1. Asegúrate de tener Node.js instalado en tu sistema.
  2. Abre una terminal o línea de comandos en la carpeta donde está tu CV.
  3. Ejecuta el archivo batch generate-pdf.bat haciendo doble clic sobre él.
  4. Espera a que se complete el proceso. El PDF se guardará como CV-Jose-Giles.pdf.

Nota: Este método requiere Chrome instalado en tu sistema y podría necesitar configuración adicional dependiendo de tu entorno.
